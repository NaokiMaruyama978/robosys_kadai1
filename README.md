## money_rateコマンド
概要
-入力された値(日本円)をドルとセント・ユーロとセント・ウォンとチョンに変換します
-半角でも全角でも変換されます
-対応してない文字や記号が入力された場合はエラー文と入力例が出ます
-何年/何月/何日/何時/何分時点のレートか表示されます

### 実行方法
#### はじめに実行権限を付与してください
```
$ chmod +x money_rate
```
